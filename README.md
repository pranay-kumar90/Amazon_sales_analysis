# Amazon Sales Analysis Project

## Overview
End-to-end data analysis project analyzing Amazon product sales data 
using SQL, Excel, and Power BI.

## Tools Used
- Excel — Data Cleaning
- MySQL — Data Analysis
- Power BI — Dashboard Creation

## Dataset
Amazon Sales Dataset from Kaggle — 1000+ products across multiple categories

## Key Insights
1. Electronics dominates product count but Home & Kitchen has better ratings
2. Products with 50%+ discount receive significantly more customer reviews
3. Mid-range products offer best value proposition for customers

## Skills Demonstrated
- Data Cleaning
- SQL Queries including Window Functions and Subqueries
- Power BI Dashboard with DAX measures
- Business Insights and Recommendations
